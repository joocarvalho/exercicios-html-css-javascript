# 🚀 Dominando HTML conectado com JavaScript

Este repositório contém uma trilha de **20 mini projetos práticos** focados no aprendizado e domínio da manipulação do DOM (Document Object Model), gerenciamento de eventos, controle de formulários e lógica de programação utilizando JavaScript puro (Vanilla JS).

A trilha foi desenhada em formato de progressão para consolidar os conceitos desde os fundamentos até o nível avançado.

## 📌 Estrutura da Trilha de Projetos

### 🟢 Fase 1: Domínio do DOM e Eventos Básicos
1. **O Contador Digital com Limites** - Manipulação de texto e escuta de cliques.
2. **Interruptor de Luz** - Gerenciamento de classes com `classList`.
3. **Gerador de Cores Aleatórias** - Manipulação de estilos e lógica com `Math.random`.
4. **O Revelador de Senhas** - Alteração dinâmica de atributos HTML.
5. **Galeria de Imagens com Miniaturas** - Manipulação de imagens e evento `mouseover`.

### 🟡 Fase 2: Formulários, Validações e Captura de Dados
6. **Validador de Senhas em Tempo Real** - Captura de eventos com `input` e validação de strings.
7. **Conversor de Temperatura Instantâneo** - Cálculos matemáticos e conversão de tipos.
8. **Calculadora de Gorjetas** - Captura de múltiplos valores e `preventDefault`.
9. **Caixa de Texto com Contador** - Manipulação de strings e limites visuais.
10. **Acordeão de Perguntas Frequentes (FAQ)** - Loops com `forEach` e seleção múltipla.

### 🔵 Fase 3: Estruturas de Dados e Dinâmica do DOM
11. **Lista de Tarefas Interativa (To-Do)** - Criação (`createElement`) e remoção de elementos.
12. **Filtro de Lista Dinâmico** - Métodos de array e busca com `.includes()`.
13. **O Cronômetro de Precisão** - Funções de tempo com `setInterval` e `clearInterval`.
14. **Sorteador de Nomes** - Manipulação de arrays e índices aleatórios.
15. **Sistema de Abas Navegáveis** - Sincronização de elementos usando `data-attributes`.

### 🟣 Fase 4: Estado da Aplicação e APIs do Navegador
16. **Lista de Desejos com Memória** - Persistência de dados locais com `LocalStorage`.
17. **O Jogo da Adivinhação de Números** - Controle de estado interno e contadores.
18. **Carrinho de Compras com Somatório** - Arrays de objetos e atualização em cadeia no DOM.
19. **Consulta de Clima Fictícia** - Simulação de requisições e leitura de objetos JSON.
20. **Validador Complexo de Cadastro** - Validações combinadas e expressões regulares (Regex).

## 🛠️ Tecnologias Utilizadas
- HTML5
- CSS3
- JavaScript (ES6+)
